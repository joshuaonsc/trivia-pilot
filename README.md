# TriviaPilot

A browser extension that automatically answers the daily **Wizard101** and **Pirate101** trivia quizzes, so you can collect your daily Crowns hands-free. Log in, start it, and solve the captchas at the end of each quiz.

> **Status: Manifest V3 port in progress.** This is the modern (MV3) successor to the v6.0.1 extension. It currently loads as an MV3 skeleton; the quiz-answering logic is being ported over phase by phase.

## Install (unpacked)

1. Clone or download this repository.
2. Open `chrome://extensions` and enable **Developer mode**.
3. Click **Load unpacked** and select the repository folder.
4. Log into Wizard101, open the extension, and start a quiz.
